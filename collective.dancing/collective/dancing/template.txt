Setup
-----

  >>> from zope import component
  >>> from zope.app.component.hooks import setSite
  >>> setSite(self.portal)

The portal tool
---------------

When installed in a Plone site, we will create a tool called
``portal_newsletters``, and therein a couple of containers:

  >>> sorted(self.portal['portal_newsletters'].objectIds())
  ['channels', 'collectors', 'composers', 'templates']

Give me a list of all templates
-------------------------------

The ``templates`` contains the list of ``ITemplate`` available.
These templatess can be retrieved through the ``ITemplateLookup``
utility, like so:

  >>> from collective.singing.interfaces import ITemplateLookup
  >>> component.getUtility(ITemplateLookup)()
  []

Let's create a template and ask for the list of templates again:

  >>> from collective.dancing.template import Template
  >>> templates = self.portal['portal_newsletters']['templates']
  >>> templates['testtemplate'] = Template('testtemplate')
  >>> component.getUtility(ITemplateLookup)()
  [<Template at /plone/portal_newsletters/templates/testtemplate>]
